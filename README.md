## Requirements

[Composer](https://getcomposer.org/)  | V2.7.2+

### Installation
1. Clone the repository:

```bash
git clone https://github.com/franmadj/laravel_shop.git
```
 ```bash
cd laravel_shop
```

 ```bash
composer install
```
 ```bash
cp .env.example .env
```
```bash
php artisan key:generate
 ```
```bash
php artisan storage:link
```
 ```bash
 php artisan migrate --seed
```
 ```bash
 npm install
```
 ```bash
 npm run prod
```
 ```bash
 php artisan serve
```
